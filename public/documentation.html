My Movie API

Welcome to the documentation for My Movie API. This API provides access to a vast library of movie information, including movie titles, release dates, ratings, and more. Through various endpoints, users can retrieve, update, delete, and add new data to the movie database. As the API evolves, this documentation will be updated with new features and endpoints. Be sure to check back often for the latest updates.

API Endpoints

Request URL Data Sent Response
GET /movies N/A Returns an array of all movies
GET /movies/:title N/A Returns data about a single movie by title, including description, genre, director, image URL, and whether it's featured or not
GET /genres/:name N/A Returns data about a genre, including its description
GET /directors/:name N/A Returns data about a director, including bio, birth year, and death year
POST /users { "Username": "example", "Password": "example", "Email": "example@example.com", "Birthday": "YYYY-MM-DD" } Returns the created user object
PUT /users/:username { "Username": "newExample", "Password": "newExample", "Email": "newExample@example.com", "Birthday": "YYYY-MM-DD" } Returns the updated user object
POST /users/:username/movies/:movieId N/A Returns the user's updated list of favorite movies
DELETE /users/:username/movies/:movieId N/A Returns the user's updated list of favorite movies after removal of a movie
DELETE /users/:username N/A Returns a message confirming the deletion of the user
